# prompt Injection (on AI browser)
## Overview
https://www.youtube.com/watch?v=ys7Vcdv4GKw&list=PLJq-63ZRPdBu38EjXRXzyPat3sYMHbIWU&index=42

- Unlike traditional cyberattacks that exploit code bugs,
- prompt injection **exploits** the AI's tendency to follow natural language instructions. 
- Hackers can hide **malicious prompts** within a webpage 
  - (e.g., hidden text or HTML comments) 
  - that trick the AI into performing unauthorized actions


## Solution:
- `Storm MCP`  | https://stormmcp.ai/apps
  - Single gateway which connects to all AI Tools (MCP servers)
  - for MCP-compliant clients
- `browserbase` | https://www.browserbase.com/contact
- Users should minimize the blast radius
- explicit human-in-the-loop confirmation for high-risk operations
