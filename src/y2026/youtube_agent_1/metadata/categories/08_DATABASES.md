# Databases & Data Storage

SQL, NoSQL, and specialized data storage technologies.

## Subcategories

### Relational Databases (SQL)
- PostgreSQL, MySQL, MariaDB, Oracle SQL
- Query optimization & indexing strategies
- ACID properties & transactions
- Normalization & schema design
- Replication & high availability
- PostgreSQL extensions (JSON, UUID, HyperLogLog)

### NoSQL Databases
- **Document Stores**: MongoDB, Firebase, CouchDB
- **Key-Value**: Redis, Memcached, DynamoDB
- **Time-Series**: InfluxDB, Prometheus, TimescaleDB
- **Graph**: Neo4j, ArangoDB
- **Search Engines**: Elasticsearch, OpenSearch, Solr
- **Vector Databases**: Pinecone, Weaviate, Milvus, ChromaDB

### Advanced Data Storage
- BigQuery, Redshift, Snowflake (Data Warehouses)
- Data Lakes & Data Lakehouses (Delta Lake, Apache Iceberg)
- HBase, Cassandra (Wide-column stores)
- Object storage (S3, GCS, Azure Blob)
- Distributed file systems (HDFS, MinIO)

### Performance & Optimization
- Query optimization & EXPLAIN plans
- Indexing strategies (B-tree, Hash, BRIN)
- Materialized views & denormalization
- Connection pooling & caching
- Sharding & partitioning strategies
- Query result caching

### Database Administration
- Backup & recovery strategies
- Disaster recovery & failover
- Capacity planning & scaling
- Database monitoring & alerting
- Security & encryption
- GDPR compliance & data deletion
